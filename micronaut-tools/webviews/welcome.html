Micronaut® Launch

Micronaut Launch is a powerful extension for using GraalVM to develop Micronaut framework applications within VS Code.

The Micronaut framework is a lightweight reactive framework that provides a solid foundation for building cloud native Java microservices.

To take your use of Micronaut further, we recommend Graal Cloud Native (GCN). Graal Cloud Native is a curated set of Micronaut framework modules designed from the ground up to be compiled ahead-of-time with GraalVM Native Image resulting in native executables ideal for microservices. GCN also provides project creation utilities and VS Code extensions for application development and deployment.
