Graal Cloud Native

Graal Cloud Native (GCN) is a full stack framework for building portable cloud-native
applications using the Java ecosystem on top of managed services in the public cloud.

OCI Support

  • 1. Create Oracle Cloud account
  • 2. Create .oci/config cloud access configuration
  • 3. Install OCI CLI needed to configure access to OKE

Tools to work with OCI DevOps

GCN development utilizes several services in OCI DevOps. To access these service following tools should be installed on development machine:

  • - Git - required for Git source code operations
  • - Docker - necessary when working locally with Docker images built by OCI pipeline
  • - Kubectl - used to access OCI applications deployed to OKE Kubernetes form local machine
